# 🔥 Phoenix Rising Landing Page

A cinematic landing page animation where **embers rise from a single point, spread across the screen, and reveal a phoenix emerging from the ashes.**

After the phoenix appears, a **Rise button** is shown. Clicking the button replaces it with the message:

**“Ready to soar”**

---

# ✨ Features

* 🔥 Ember particle animation starting from **one origin point**
* 🌌 Gradual **screen-wide ember spread**
* 🐦 **Phoenix image reveal**
* ✨ Phoenix **glow aura effect**
* 🕊 Floating phoenix animation
* 🎬 Cinematic landing page experience
* 🖱 Button interaction replacing itself with a message

# 🚀 How to Run

1. Clone the repository

```
git clone https://github.com/yourusername/phoenix-landing-page.git
```

2. Open the project folder

```
cd phoenix-landing-page
```

3. Open the file

```
index.html
```

in your browser.

No build tools required — this project runs with **pure HTML, CSS, and JavaScript.**

# ⏱ Animation Duration

The ember animation duration can be configured in the code:

```javascript
let duration = 60000; // demo time
```

Values are in **milliseconds**.

Examples:

| Value  | Duration   |
| ------ | ---------- |
| 12000  | 12 seconds |
| 60000  | 1 minute   |
| 900000 | 15 minutes |
